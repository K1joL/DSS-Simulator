# ПАК КЦПС Simulator

Итоговый starter-проект для исследования трафика, потерь, RTO/RPO и сравнения алгоритмов резервирования функциональных контекстов.

## Что внутри

- ядро симуляции на TypeScript,
- UI на React + Vite,
- batch/sweep эксперименты,
- 3 встроенных алгоритма (`delta`, `full-sync`, `lazy-checkpoint`),
- JSON import/export конфигурации,
- CSV export batch-результатов,
- набор тестов на Vitest.

## Быстрый запуск

### 1. Установить Node.js
Рекомендуется Node.js 20+.

### 2. Установить зависимости
```bash
npm install
```

### 3. Запустить UI
```bash
npm run dev
```

После запуска Vite покажет локальный адрес, обычно `http://localhost:5173`.

### 4. Собрать production build
```bash
npm run build
npm run preview
```

### 5. Запустить тесты
```bash
npm test
```

## Как пользоваться

### Конфигурация
- На экране **Конфигурация** настраиваются seed, длительность, шаг, параметры шины, алгоритм и параметры алгоритма.
- Можно импортировать и экспортировать JSON-конфигурацию.

### Симуляция
- Кнопки: `Старт`, `Пауза`, `Шаг`, `До конца`, `Сброс`.
- У ВМ есть ручные действия `Fail` и `Recover`.

### Batch
- На экране **Batch** задаются seeds, список алгоритмов и sweep-параметры.
- Есть экспорт `batch-runs.csv`, `batch-ranking.csv` и `batch-result.json`.

## Ограничения текущей версии

- Это исследовательский прототип, а не промышленный сетевой эмулятор.
- Трафик считается агрегированными пакетами за шаг, а не покадрово.
- UI ориентирован на анализ и отладку, а не на финальную визуализацию.
